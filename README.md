###### 运行要求

1. 安卓 8.0+ 
2. 已获取Root权限 (Magisk 23.0+)
3. 64位 处理器 (推荐)

###### 简介

Stryker 是新一代的移动渗透测试应用程序。它将帮助小白在没有特殊技能和知识的情况下测试他们网络设备的常见漏洞。并且还为您提供了添加利用漏洞的脚本和新的功能！
